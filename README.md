# agentexpert-io

The definitive resource for building AI agents — roadmaps, patterns, case studies and tips.

Built with Astro + Keystatic by [Zahiruddin Tavargere](https://zahere.com)

## Deploy to Vercel

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/zahere-dev/agentexpert-io)

## Local Dev

```bash
npm install
npm run dev
```

Visit `http://localhost:4321/keystatic` to manage content.
